# Sistema de Gestión de Inventario y Ventas para Supermercado Ara

## Descripción General

Este es un proyecto **académico** desarrollado en **C# con Windows Forms** y conectado a una **base de datos Oracle**.

El objetivo principal es simular el funcionamiento básico de un supermercado, abarcando las áreas de:
* Inventario
* Productos
* Ventas
* Proveedores
* Manejo de clientes

> **Nota:** Este sistema no está diseñado para usarse en producción real; funciona únicamente como herramienta académica y demostrativa.

---

## Roles de Usuario

La aplicación está diseñada para manejar dos roles principales:

| Rol | Descripción | Funcionalidades Clave |
| :--- | :--- | :--- |
| **Cliente** | Puede interactuar con la tienda y realizar compras. | Explorar productos, ver categorías, usar el carrito, realizar compras y gestionar su cuenta. |
| **Administrador** | Puede gestionar y controlar toda la operativa del negocio. | Gestionar productos, categorías, compras, ventas y revisar información del inventario. |

---

## Tecnologías Utilizadas

Las herramientas y tecnologías principales usadas en el desarrollo son:

* **Lenguaje:** C#
* **Interfaz de Usuario:** Windows Forms (Desarrollado en Visual Studio 2022)
* **Base de Datos:** Oracle Database (Gestionado con SQL Developer)
* **Lógica de Base de Datos:** PL/SQL (Uso de paquetes, procedimientos y funciones)
* **Control de Versiones:** Git y GitHub

---

## Características Principales

El sistema incluye las siguientes funcionalidades destacadas:

* Pantalla de inicio con **Login** y **Registro** de usuarios.
* **Catálogo de Productos** organizado por categorías.
* **Carrito de Compras** completamente funcional.
* Módulo de **Administración de Inventario** exclusivo para el rol administrador.
* Registro de proveedores, manejo de compras, ventas y pedidos.
* **Actualización automática de stock** al realizar ventas o registrar compras.
* Diseño sencillo y fácil de usar.
* Incluye **Scripts SQL completos** para la creación y configuración inicial de la base de datos.
